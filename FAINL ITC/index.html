Addendum

CSS Styles: Using a CSS grid layout for the game board allows for an organized display of squares. The is-off class helps differentiate the black squares.

JavaScript Logic: The JavaScript code dynamically creates the game board, toggles squares and their neighboring squares, and checks for a win condition when all squares are black.

Random Solvable Configuration: The approach here toggles a random selection of squares initially, ensuring a solvable configuration. Toggling any square will also toggle its neighboring squares, thus shaping the solvability of the game.

Challenges: Ensuring the initial configuration is solvable was a primary challenge. By toggling random squares and their neighbors, a solvable state is more likely achieved.

Insights Gained: This approach showcases how a simple game like Lights Out can be built using JavaScript and CSS. The toggle mechanism applied to squares and their neighbors significantly simplifies the logic behind the solvability of the puzzle.
