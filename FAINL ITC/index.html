Lights Out


Addendum

CSS Styles: Using a CSS grid layout for the game board allows for an organized display of squares. The is-off class helps differentiate the black squares.

JavaScript Logic:

Global Variables
boardSize: Represents the size of the game board. In this code, it's set to 5, indicating a 5x5 grid.
board: An array used to keep track of the state of each square on the game board. It represents whether a square is 'on' (true) or 'off' (false). Initially, all squares are set to 'on' (true).

createBoard() Function
createBoard() function initializes the game board by creating a grid of squares inside an element with the ID gameBoard.
It loops boardSize * boardSize times to create the squares.
Each square is represented by a <div> element with the class square, and it's given a dataset attribute index to store its corresponding index in the board array.
Event listeners are added to each square so that when clicked, the toggleSquare() function is invoked with the index of the clicked square.

Generating a Solvable Random Board
After creating the board, the code simulates random clicks to generate a solvable random initial state.
It loops through boardSize * boardSize times and toggles random squares using the toggleSquare() function.

toggleSquare(index) Function
This function is called when a square is clicked.
It toggles the state of the clicked square and its adjacent squares by changing their CSS classes and updating the board array.
The clicked square's state is toggled by updating the board array at the clicked index and toggling the is-off class to change its appearance.
Adjacent squares' states are also toggled by getting their indices using the getAdjacentIndices() function and updating their states and classes accordingly.

getAdjacentIndices(index) Function
This function returns an array of indices of adjacent squares to a given square index.
It checks whether the clicked square is at the edges of the board and adds the indices of adjacent squares accordingly (up, down, left, right).

checkWin() Function
Checks if all squares are turned off (false) by using the every() method on the board array.
If all squares are off, it triggers an alert message indicating that the game has been won.

resetBoard() Function
Resets the board array and recreates the game board by calling createBoard().

Initialization
Calls createBoard() to start the game when the script is loaded.

Overall, this code creates a Lights Out puzzle game where clicking a square toggles its state and the state of adjacent squares. The goal is to turn off all the squares to win the game. The initial board configuration is randomly generated and guaranteed to be solvable.

Random Solvable Configuration: The approach here toggles a random selection of squares initially, ensuring a solvable configuration. Toggling any square will also toggle its neighboring squares, thus shaping the solvability of the game.

Challenges: Ensuring the initial configuration is solvable was a primary challenge. By toggling random squares and their neighbors, a solvable state is more likely achieved.

Insights Gained: This approach showcases how a simple game like Lights Out can be built using JavaScript and CSS. The toggle mechanism applied to squares and their neighbors significantly simplifies the logic behind the solvability of the puzzle.
